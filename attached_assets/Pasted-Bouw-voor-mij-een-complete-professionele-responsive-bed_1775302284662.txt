Bouw voor mij een complete, professionele, responsive bedrijfswebsite voor een Nederlands schoonmaakbedrijf met de naam “Agion Cleaning”.

BELANGRIJK:
- Lever een volledig werkende website op, niet alleen een mockup.
- Schrijf de hele website in het Nederlands.
- Maak het design modern, strak, professioneel, betrouwbaar, conversiegericht en geschikt voor een schoonmaakbedrijf.
- Laat de uitstraling een mix zijn van:
  1) toegankelijk en duidelijk voor particuliere klanten
  2) professioneel en zakelijk sterk voor zakelijke klanten
- Gebruik de stijl van moderne schoonmaak- en facilitaire bedrijven als inspiratie, maar kopieer geen bestaande website letterlijk.
- De website moet geschikt zijn voor mobiel, tablet en desktop.
- Gebruik een nette, rustige kleurencombinatie die past bij schoonmaak, frisheid, vertrouwen en professionaliteit.
- Bouw herbruikbare componenten en nette projectstructuur.
- Zorg dat het project direct goed draait in Replit.

TECHNISCHE KEUZE:
- Kies een moderne, stabiele stack die goed werkt in Replit.
- Voorkeur: React + TypeScript + Tailwind CSS + nette componentstructuur.
- Als Next.js in Replit stabieler is, gebruik dan Next.js met TypeScript en Tailwind.
- Gebruik een professionele folderstructuur.
- Voeg duidelijke comments toe waar configuratie of content later aangepast moet worden.
- Maak de code schoon, leesbaar en production-ready.

CONTACTFORMULIER:
- Maak een volledig werkend contactformulier.
- Velden:
  - Naam
  - E-mailadres
  - Telefoonnummer
  - Type klant (Particulier / Zakelijk)
  - Onderwerp / Gewenste dienst
  - Bericht
- Het formulier moet server-side verwerkt worden.
- Bij verzending moet de klant direct een e-mailmelding ontvangen.
- Gebruik hiervoor een nette backend-oplossing met environment variables.
- Gebruik bijvoorbeeld SMTP/Nodemailer of een vergelijkbare nette oplossing.
- Voeg duidelijke placeholders toe voor:
  - CONTACT_EMAIL
  - SMTP_HOST
  - SMTP_PORT
  - SMTP_USER
  - SMTP_PASS
- Laat in de code duidelijk zien waar deze env vars moeten worden ingesteld.
- Voeg nette success- en foutmeldingen toe in het formulier.
- Voeg basisvalidatie toe.
